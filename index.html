电量分流计算器

使用说明

功能：根据电力参数计算放电时间，并通过组合使用 1/2分流、1/3分流 和 2/3分流 来精确匹配目标输出效率。

输入：

  • 目前消耗(x)：热能池缺电时游戏上方数字 x/y 中的 x
  • 目前生产(y)：热能池缺电缺电时游戏上方数字 x/y 中的 y

电池容量：z

计算逻辑：

  • 电池总容量：100000
  • 充电时每秒增加：y + z - x
  • 充电时间固定：40秒
  • 40秒充电量 = min((y - x) × 40, 100000)
  • 掉电速率 = x - y
  • 最佳掉电时间 = 40秒充电量 / 掉电速率
  • 目标效率 = 最佳掉电时间 + 40秒

输出：具体的分流方案和实际效率（基础速度为 2秒/个）。

注意： 本计算器基于从仓库拉取产线进行计算，而非从电池制造设备拉出。仓库产线具有最高运输效率和更好的可操控性。
